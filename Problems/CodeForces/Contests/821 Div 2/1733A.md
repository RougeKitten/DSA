# Approach
- Since at the end after all the operations are done, we need to select a set of k consecutive elements such that the sum of this subarray is maximum, we stored the maximum value present at ```i mod k``` and then take a sum all all these values (mod of any number with k will give us values which lie in the range of 0-k - 1, i.e k values in total)
