# Fast Exponentiation

```
a^n = (a^n/2)*(a^n/2) if n is even
	= a*(a^n/2)*(a^n/2) if n is odd
```

## Code
```cpp
#include<bits/stdc++.h>
using namespace std;

int solve(int a, int n){
	if(n == 0)
		return 1;
	if(n == 1)
		return a;
	int x = solve(a, n/2);
	if(n&1)
		return a*x*x;
	else
		return x*x;
}

int main(){
	int a, n;
	cin >> a >> n;

	cout << solve(a, n) << endl;
}
```
